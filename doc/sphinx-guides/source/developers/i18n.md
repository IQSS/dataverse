# Internationalization (i18n)

## Intro

This section is for developers working on internationalizing Dataverse. If you are running Dataverse and want to configure support for a language other than English, please see {ref}`i18n` in the Installation Guide.

## Creating languages.zip

This script creates a languages.zip file in the format expected by the {ref}`load-langs` step.

{download}`make-languages-zip.sh <../../../../scripts/i18n/make-languages-zip.sh>`
