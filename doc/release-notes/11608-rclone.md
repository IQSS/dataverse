### Rclone Support

Rclone ("rsync for cloud storage") is a command-line program to sync files and directories to and from different cloud storage providers. As of version 1.70 Rclone supports Dataverse. See [the announcement](https://groups.google.com/g/dataverse-community/c/IoxF85arJhA/m/QFD6vP-bBQAJ), [the guides](https://dataverse-guide--11609.org.readthedocs.build/en/11609/admin/integrations.html#rclone), #11608, and #11609.
